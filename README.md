# midnight-dex-contract

Midnight AMM DEX contract, written in Compact.

This repository follows the layout recommended in the Midnight tutorial:

- `contracts/` — Compact smart contracts for the DEX (currently includes `AMM.compact`).
- `proofs/` — Plonk/zk proof assets or proving keys generated during development.
- `scripts/` — Helper scripts for compiling contracts, generating proofs, and deployment utilities.
- `clients/` — Front-end or backend clients that interact with the contract APIs.
- `tests/` — Contract and integration test suites.
- `docs/` — Additional project documentation produced while working through the tutorial.

Populate each folder as you build out the tutorial steps.
